# Инструкция по запуску

## Быстрый старт

1. **Установите зависимости:**
   ```bash
   cd mobile
   flutter pub get
   ```

2. **Настройте API URL:**
   
   Откройте `lib/config/api_config.dart` и измените `baseUrl`:
   
   - **Для Android эмулятора** (уже настроено): `http://10.0.2.2:8000`
   - **Для реального устройства**: `http://<IP_вашего_компьютера>:8000`
     - Чтобы узнать IP: `ipconfig` (Windows) или `ifconfig` (Linux/Mac)
     - Пример: `http://192.168.1.100:8000`
   - **Для продакшена**: укажите URL вашего сервера

3. **Убедитесь, что бэкенд запущен:**
   ```bash
   cd ../backend
   # Запустите ваш бэкенд
   ```

4. **Запустите приложение:**
   ```bash
   flutter run
   ```

## Тестирование

1. Запустите Android эмулятор или подключите реальное устройство
2. Убедитесь, что бэкенд доступен по указанному адресу
3. Войдите с существующими учетными данными или создайте новый аккаунт через веб-версию

## Возможные проблемы

### Ошибка подключения к API

- Проверьте, что бэкенд запущен
- Проверьте URL в `api_config.dart`
- Для реального устройства убедитесь, что телефон и компьютер в одной сети
- Проверьте, что файрвол не блокирует подключение

### Ошибка "Network is unreachable"

- Убедитесь, что эмулятор/устройство имеет доступ к интернету
- Проверьте настройки сети в эмуляторе
